About Text Case Converter

This tool helps you convert text between different cases instantly. Simply enter your text, select the desired case type, and get the converted result.

Available Case Types:
  • UPPERCASE - Converts all letters to uppercase
  • lowercase - Converts all letters to lowercase
  • Title Case - Capitalizes the first letter of each word
  • Sentence case - Capitalizes the first letter of each sentence
  • camelCase - Removes spaces and capitalizes the first letter of each word except the first
  • PascalCase - Removes spaces and capitalizes the first letter of each word
  • snake_case - Replaces spaces with underscores
  • kebab-case - Replaces spaces with hyphens
  • iNVERT cASE - Inverts the case of each letter
  • aLtErNaTiNg CaSe - Alternates between uppercase and lowercase letters
How to Use
  1. Enter or paste your text in the input box
  2. Select the desired case type from the dropdown
  3. Click "Convert" or wait for automatic conversion
  4. Copy the converted text using the "Copy" button
