About This Tool

Generate a robots.txt file for your website to control search engine crawlers. A robots.txt file tells search engines which pages or files they can or cannot request from your site.

Features:

  • Generate standard robots.txt file
  • Add custom disallow rules
  • Include sitemap reference
  • Copy to clipboard
  • Download as .txt file
