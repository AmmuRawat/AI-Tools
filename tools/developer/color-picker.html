About This Tool

This Color Picker tool helps you select, convert, and work with colors in various formats. Features include:

  • Interactive color selection
  • Multiple color format support (HEX, RGB, HSL, CMYK)
  • Color scheme generation
  • Copy color values to clipboard
  • Random color generation

Color Formats:

  • HEX: #RRGGBB format
  • RGB: Red, Green, Blue values (0-255)
  • HSL: Hue, Saturation, Lightness values
  • CMYK: Cyan, Magenta, Yellow, Key (Black) values
